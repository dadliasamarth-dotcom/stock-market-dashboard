# Stock Market Dashboard 📈

Developed a Stock Market Dashboard with real-time stock tracking, interactive charts, search functionality, and portfolio analytics.

---

## Project Overview

This project helps users track multiple stocks, analyze price trends, and monitor portfolio performance through an interactive dashboard.

The application fetches stock data, stores it, processes analytics, and displays it through a modern frontend dashboard.

---

## Features

✅ Multi-stock tracking  
✅ Interactive stock price charts  
✅ Search stock functionality  
✅ Portfolio KPI cards  
✅ Market watch cards  
✅ Buy/Sell buttons  
✅ Risk score display  
✅ Backend API integration  
✅ Responsive UI dashboard


### Frontend Setup

```bash
cd stock-frontend
npm install
npm start
```
